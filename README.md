# kunal-portfolio (Solid)

```bash
npm install
npm run dev
```

Then open http://localhost:5173

**SolidJS notes** (since you came from React):
- Use `class` not `className`
- State values are functions: write `scrolled()` not `scrolled` to read them
- Lifecycle: `onMount` instead of `useEffect`

Everything else is the same. Drop your `cv.pdf` into `public/` to make the CV link work.
